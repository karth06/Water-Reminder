# Changelog

All notable changes to the "Water Reminder - Retro Edition" extension will be documented in this file.

## [1.0.0] - 2025-11-16

### Added
- Initial release
- Customizable countdown timer for water reminders
- Retro arcade-themed webview with neon aesthetics
- Daily water intake tracking
- Status bar integration
- Sidebar tree view
- Sound notifications
- Persistent state across sessions
- Auto-start timer option
- Configurable daily goals
- Multiple views: sidebar and full retro panel

### Features
- Timer with customizable intervals (1-240 minutes)
- Visual progress tracking with animated bars
- CRT monitor effects and pixel art animations
- System sound notifications
- State persistence using VS Code global storage
- Daily progress auto-reset at midnight
- Snooze functionality
- Achievement messages for goal completion
